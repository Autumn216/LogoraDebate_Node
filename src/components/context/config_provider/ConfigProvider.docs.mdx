import { ConfigProvider } from './ConfigProvider';
import { useConfig } from './useConfig';
import { useRoutes } from './useRoutes';

ConfigProvider is a context component to store configuration variables and routes.

To use the context, first wrap your application with _ConfigProvider_. Then, use _useConfig_ and _useRoutes_ hooks to get config and routes.

### Component usage
```js
const ComponentWithConfig = () => {
	const config = useConfig();
	const routes = useRoutes();

	return <div>Hello world</div>;
}

<ConfigProvider config={layout: "compact"}>
    <ComponentWithConfig />
</ConfigProvider>
```
