import { FacebookLoginButton } from './FacebookLoginButton';
import { MemoryRouter } from "react-router";

A button component to login to Facebook with OAuth2.

When clicking on the button, a popup will appear that will ask the user to login to Facebook. User will be
redirected to _redirectUri_ afterwards if login is successful, with authorization code.

Two scopes are requested :
- email
- public_profile

### Component usage
```js
import FacebookLoginButton from '@logora/debate.auth.facebook_login_button';
```
```js
    <MemoryRouter>
        <FacebookLoginButton 
            text={"Sign in with Facebook"}
            facebookClientId={"facebook-client-id"}
            redirectUri={"https://auth.redirect/}
            onCode={(code) => console.log(code)}
            onClose={closeCallbackFunction}
        />
    </MemoryRouter>
```

### Properties
| **Name**      | **Type** | **Required** | **Default** | **Description** |
| ----------- | ----------- | :-----------: | ----------- | ----------- |
| <font color='#4646c6'>text</font> | <font color="#397300">string</font> | &check; | - | <font color="grey">*button text*</font> |
| <font color='#4646c6'>facebookClientId</font> | <font color="#397300">string</font> | &check; | - | <font color="grey">*facebook oauth2 client ID*</font> |
| <font color='#4646c6'>redirectUri</font> | <font color="#397300">string</font> | &check; | - | <font color="grey">*oauth2 redirect URI*</font> |
| <font color='#4646c6'>onCode</font> | <font color="#397300">function</font> | &check; | - | <font color="grey">*callback triggered if auth is successful. onCode takes one argument, which is the oauth2 authorization code*</font> |
