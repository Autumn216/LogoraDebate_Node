import { DialogBox } from './DialogBox';

This component must be wrapped by IntlProvider. The component position is provided by props around it's children.

It displays a title, a content and a button to close it.

### Component usage
```js
import DialogBox from '@logora/debate.dialog.alert_dialog';
```
```js
<IntlProvider locale='en'>
    <DialogBox 
        isBottom
        isPoints 
        titleKey={"info.debates"} 
        contentKey={"info.debate.first_time"}
        isHidden={false}
    >
        <span>React child</span>
    </DialogBox>
</IntlProvider>
```
