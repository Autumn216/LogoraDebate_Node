import { UserBox } from './UserBox';

A component that renders a user box. It renders a profile image, a full name and a description.

### Component usage
```js
import { UserBox } from '@logora/user.user_box';
```
```js
<BrowserRouter>
    <IntlProvider locale="en">
        <ConfigProvider routes={{...routes}}>
            <UserBox user={user} />
        </ConfigProvider>
    </IntlProvider>
</BrowserRouter>
```

### Properties
| **Name**      | **Type** | **Required** | **Default** | **Description** |
| ----------- | ----------- | :-----------: | ----------- | ----------- |
| <font color='#4646c6'>user</font> | <font color="#397300">object</font> | &check; | - | <font color="grey">*a user object gathering all the user infos*</font> |
