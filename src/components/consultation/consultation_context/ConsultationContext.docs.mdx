import { ConsultationContext } from './ConsultationContext';

A component that renders a consultation context header. It renders an image, a full title, a description and a vote goal about the consultation.

### Component usage
```js
import { ConsultationContext } from '@logora/debate.consultation.consultation_context';
```
```js
    const consultation = {
        id: 19,
        slug: "festivals-quelle-serait-votre-programmation-de-reve",
        title: "Festivals: quelle serait votre programmation de rêve ?",
        description: "Consultation de rêve",
        description_url: "",
        created_at: "2022-11-24T15:20:51.727Z",
        ends_at: "2023-10-30T00:00:00.000Z",
        vote_goal: 1000,
        total_votes: 2,
        total_participants: 4,
        proposals_count: 53,
        image_url: "https://dfrx2oay1w3r9.cloudfront.net/uploads/consultations/banner_8e9d9ab1cdf82a80c2315488195e1fc8.png",
        direct_url: "https://test.logora.fr/#/consultation/festivals-quelle-serait-votre-programmation-de-reve",
        tags: [
            {
                id: 37,
                name: "musique",
                taggings_count: 0,
                display_name: "musique"
            },
        ],
    }
    
    <IntlProvider locale="en">
        <ConsultationContext consultation={consultation} />
    </IntlProvider>
```

### Properties
| **Name**      | **Type** | **Required** | **Default** | **Description** |
| ----------- | ----------- | :-----------: | ----------- | ----------- |
| <font color='#4646c6'>consultation</font> | <font color="#397300">object</font> | &check; | - | <font color="grey">*a consultation object gathering all the consultation infos*</font> |
| <font color='#4646c6'>disabled</font> | <font color="#397300">boolean</font> | &check; | - | <font color="grey">*true if consultation is ended*</font> |
